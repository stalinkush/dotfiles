# Dotfiles

These dotfiles auto-install [scc](https://github.com/boyter/scc) in every GitHub Codespace.

## Features
- Installs `scc` into `~/bin`
- Adds `~/bin` to PATH
- Provides optional aliases in `.bashrc`

## Usage
1. Link this repo in GitHub → Settings → Codespaces → Dotfiles.
2. Enable "Automatically install dotfiles".
3. Open any Codespace — `scc` will be ready!
